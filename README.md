## Quickstart

To build project create `Secrets.swift` file with two variables in it:
```swift
let GOOGLE_OAUTH2_CLIENT_ID = "<YOUR_GOOGLE_OAUTH2_CLIENT_ID>"
let GMS_API_KEY = "<YOUR_GOOGLE_MAPS_KEY>"
```
